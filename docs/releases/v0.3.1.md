# RenderNorth Display Switcher v0.3.1

This usability release makes installation, launching, updates, and Stream Deck setup straightforward while preserving the verified display-switching engine.

## Recommended Download

**RenderNorth-Display-Switcher-Setup-v0.3.1.exe**

For most users, download and run the Setup EXE. It installs the normal GUI, creates Windows shortcuts, and supports optional in-app updates.

## Advanced

**RenderNorth-Display-Switcher-v0.3.1-win-x64-Portable-Manual-Updates.zip**

The portable edition is intended for testing and advanced use. Extract the complete ZIP and replace it manually when updating.

## Updater Support Files

Files such as `.nupkg`, `RELEASES`, and `releases.win.json` are used by the automatic updater. They are not intended for manual installation.

## Changes

- Added stable installed Start-menu shortcuts for Game Mode and Script Mode Stream Deck actions.
- Explicitly creates the main Start-menu and Desktop shortcuts.
- Keeps installed profiles and logs outside Velopack's replaceable application-content directory.
- Adds best-effort migration for profiles created by earlier installed builds.
- Clarifies Setup, portable, first-run, update, and Stream Deck instructions.

The native display-switching implementation is unchanged.
