# RenderNorth Display Switcher v0.3.2

This launcher hotfix preserves the verified display-switching engine and corrects installed Game Mode and Script Mode activation while the GUI is open.

## Recommended Download

**RenderNorth-Display-Switcher-Setup-v0.3.2.exe**

For most users, download and run the Setup EXE. Existing installed v0.3.1 users can use **Check for Updates**.

## Advanced

**RenderNorth-Display-Switcher-v0.3.2-win-x64-Portable-Manual-Updates.zip**

The portable edition requires manual replacement.

## Updater Support Files

The `.nupkg`, `RELEASES`, and `releases.win.json` files are updater support assets and are not intended for manual installation.

## Fixes

- Installed `RenderNorthGameMode.exe` and `RenderNorthScriptMode.exe` now work while the normal GUI is open.
- Each launcher resolves `RenderNorthDisplaySwitcher.exe` beside itself using `AppContext.BaseDirectory`.
- Launcher failures and nonzero main-application exit codes are written to `launcher-YYYYMMDD.log`.
- Launchers remain silent with no console window or success popup.

The native Windows display-layout engine is unchanged.
