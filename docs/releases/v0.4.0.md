# RenderNorth Environments v0.4.0 — Early Access

RenderNorth Environments lets you save and activate named Windows display workspaces for gaming, streaming, development, presentations, and more.

## Highlights

- Named environments with stable GUIDs
- Native Windows display capture and activation
- Human-readable environment previews and categories
- Silent Stream Deck shortcuts
- Legacy Game Mode and Script Mode migration
- Settings, diagnostics, validation, and rollback protection
- Velopack installer and update compatibility

## Early Access

The first environment capability manages display configurations. Future capabilities may expand beyond displays, but no dates or feature promises are made.

Display environments may need to be recaptured after Windows updates, GPU-driver updates, docking changes, or monitor re-enumeration.

For most users, download the Windows Setup EXE. The Portable ZIP is intended for advanced users and requires manual updates.
