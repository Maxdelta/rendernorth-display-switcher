# RenderNorth Environments Marketing Draft

> v0.4 is not public. Do not use this copy as a release announcement until installed migration testing and owner approval are complete.

## Elevator pitch

RenderNorth Environments lets your Windows PC switch its workspace to match what you are doing.

## Product statement

Save environments for Gaming, Streaming, Development, Editing, Presentation, Work, Travel, or any custom workflow. Activate the right workspace from the desktop app, a command, or a stable Stream Deck shortcut.

Version 0.4 begins with native Windows display configuration. It preserves exact display topology, validates monitor identities, verifies activation, and attempts rollback after failure. The modular foundation is designed for future real capabilities without exposing unfinished controls today.

## Feature list

- Multiple named environments with permanent GUID identity
- Editable names, descriptions, controlled icons, and categories
- Native Windows display capture and activation
- Current-environment detection
- Transactional lifecycle with rollback
- Stable rename-safe Desktop, Start menu, and custom shortcuts
- Silent Stream Deck and command-line activation
- Automatic legacy Game Mode and Script Mode migration
- Local data and logs with no telemetry or account
- In-place compatibility with the existing installer and updater identity

## Short announcement draft

RenderNorth Environments is the next evolution of RenderNorth Display Switcher: a Windows workspace launcher that adapts your PC to Gaming, Streaming, Development, Editing, Presentation, Work, and more. The v0.4 preview begins with native display environments, stable Stream Deck shortcuts, automatic legacy migration, and the proven rollback-aware display engine.

## Demo outline

1. Open RenderNorth Environments and show the Current Environment card.
2. Show migrated Game Mode and Script Mode environments.
3. Capture the active display setup into a new Development environment.
4. Rename it and confirm its permanent ID remains unchanged.
5. Activate another environment and show detection update.
6. Create a Start menu shortcut and add it to Stream Deck.
7. Activate through Stream Deck without visible application UI.
8. Explain that Displays are the first capability in a modular workspace platform.

## Required proof before publication

- Public v0.3.2 to v0.4 installed update succeeds.
- Legacy data backup and migration are confirmed.
- Both migrated display configurations activate on physical hardware.
- Generated shortcut survives environment rename and application update.
- Owner verifies physical Stream Deck operation.
- New screenshots are captured from the installed v0.4 application without personal information.
