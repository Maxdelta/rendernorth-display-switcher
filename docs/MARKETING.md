# RenderNorth Display Switcher Marketing Kit

## Elevator pitch

RenderNorth Display Switcher gives dual-PC creators one-click, Stream Deck-ready control over which monitor is sent to their capture card while keeping private screens private.

## Feature list

- One-click Game Mode and Script Mode switching
- Silent Stream Deck launcher executables
- Native Windows 11 display configuration APIs
- Stable monitor identity matching
- Rollback and post-switch verification
- Current-profile and last-switch status
- Local diagnostic logging
- Optional installed-app updates
- Portable Windows x64 edition
- No OBS, Streamlabs, or DisplayFusion dependency
- No accounts, ads, analytics, or telemetry

## Release announcement

RenderNorth Display Switcher is now available for Windows 11. Built for dual-PC creators, it saves the display layouts you have already configured and restores Game Mode or Script Mode with one click—or silently from Stream Deck. Route your main gaming display to the capture card, switch the capture to a second monitor, and keep a private script screen out of the feed.

The first public RenderNorth desktop application includes an installer with optional updates, a portable edition, dedicated Game and Script launchers, rollback protection, local logs, and complete source under the MIT License.

Download: https://github.com/Maxdelta/rendernorth-display-switcher/releases/latest

## Reddit

**Title:** I built a free Windows display switcher for dual-PC streaming and Stream Deck

I use a dual-PC setup with an Elgato HD60 X and needed a reliable way to switch which physical monitor was duplicated to the capture output. Windows Display Settings worked, but it was too awkward during a stream.

RenderNorth Display Switcher lets you configure two layouts in Windows—Game Mode and Script Mode—save them, and restore either one with a click. It also includes silent launcher executables for Stream Deck, so no app window or success popup interrupts the feed.

It uses native Windows display APIs, validates monitor identities, attempts rollback on failure, and keeps local logs. It does not depend on OBS, Streamlabs, or DisplayFusion. Windows 11 x64 only; my verified setup uses an Elgato HD60 X, so other capture hardware should be tested before going live.

Source and downloads: https://github.com/Maxdelta/rendernorth-display-switcher
