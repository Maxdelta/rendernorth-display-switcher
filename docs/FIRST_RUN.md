# First Run Guide

1. Open **RenderNorth Display Switcher** from the Windows Start menu.
2. Configure the gaming layout in **Settings > System > Display**, then select **Save Current Layout as Game Mode**.
3. Configure the private-script layout, then select **Save Current Layout as Script Mode**.
4. Test both **Activate** buttons while watching the capture preview.
5. In Stream Deck, add **System > Open** actions for the installed Start-menu shortcuts:
   - **RenderNorth Display Switcher - Game Mode**
   - **RenderNorth Display Switcher - Script Mode**
   - Find both shortcuts under `%APPDATA%\Microsoft\Windows\Start Menu\Programs`.
6. Use **Check for Updates** in the GUI when you want to install a newer release.

The Game and Script shortcuts are silent and survive application updates. A brief black screen while Windows renegotiates the capture signal is normal.
