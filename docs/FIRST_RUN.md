# RenderNorth Environments First Run

1. Open **RenderNorth Environments** from the Windows Start menu.
2. If updating from Display Switcher v0.3.2, confirm Game Mode and Script Mode appear as migrated environments.
3. Select an environment to activate it, or choose **Capture Current Setup** to save the active Windows display workspace.
4. Enter a unique name and optional description, then choose a built-in icon and category.
5. Save and activate the environment. Confirm the Current Environment card detects it.
6. From the environment’s More menu, create a stable Start menu or Desktop shortcut.
7. Add that shortcut to Stream Deck with **System → Open**.
8. Test display and capture-card behavior before relying on it during a live production.

Generated shortcuts use the permanent environment GUID and remain valid after application updates and environment renames.

Legacy Game and Script shortcuts remain supported during v0.4 migration. Their underlying legacy aliases are independent of the visible environment names.
